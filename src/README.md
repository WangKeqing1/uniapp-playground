## 📑 模块说明

`src` 目录是项目的源码核心基座，承载了所有跨端页面的入口、应用级生命周期以及多端打包配置。

## 📍 关键文件 & 入口

| 文件            | 描述                                                    |
| --------------- | ------------------------------------------------------- |
| `main.ts`       | 应用程序启动的全局入口文件。                            |
| `App.vue`       | 根组件，配置应用级生命周期和全局状态注入。              |
| `manifest.json` | 配置应用名称、appid、logo、版本等打包多端时的基础信息。 |
| `pages.json`    | 页面路由、导航条、选项卡等全局设置。                    |
| `uni.scss`      | 全局的 SCSS 样式变量预设文件。                          |

## 🔗 依赖与子模块

- [页面模块 (`pages`)](./pages/README.md)：存放所有业务页面。
- `static`：存放静态公共资源（图片、字体等）。
